Test de velocidad

Mide la latencia y la velocidad de descarga de tu conexión usando archivos alojados en esta misma página.

Latencia -- ms
Descarga -- Mbps
Jitter -- ms

Presiona “Iniciar test” para comenzar.

Importante: en GitHub Pages este test puede medir latencia y descarga. La velocidad de subida no se puede medir de forma real sin un servidor externo que reciba datos.

Archivos necesarios: sube junto a este index.html los archivos speedtest_10mb.bin, speedtest_50mb.bin y speedtest_100mb.bin.

Si solo quieres una versión simple, basta con subir speedtest_10mb.bin y usar la opción de 10 MB.
